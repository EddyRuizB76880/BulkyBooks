# CRUD_MVC
During this section, a small CRUD web application was implemented using Entity Framework.
The following topics were reviewed:
- Entity Framework.
- Tag helpers.
- MVC Pipeline.
- Views, controllers y models usando .NET
- A little bit of Bootstrap.
- Shared views.
